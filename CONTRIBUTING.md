# Contributing to Deploy Buttons

Thank you for your interest in contributing to Deploy Buttons! This document provides guidelines and instructions for contributing.

## How to Contribute

### Reporting Issues

If you find a bug or have a suggestion, please open an issue using the appropriate template:
- **Button Request**: Use the "Request New Deploy Button" template to propose a new deploy button
- **Bug Report**: Open a regular issue to report problems with existing buttons

### Adding New Deploy Buttons

1. **Fork the repository** and create a new branch for your button
2. **Create the SVG file** following these guidelines:
   - File name format: `deploy-{platform-name}.svg` (lowercase, hyphens for spaces)
   - SVG should be optimized and valid
   - Recommended dimensions: 200x50px or similar aspect ratio
   - Use consistent styling with existing buttons
3. **Add the button to the repository**:
   - Place the SVG file in the `svgs/` directory
   - Update the README.md with the new button section
   - Follow the existing format for consistency
4. **Test your button**:
   - Verify the SVG displays correctly
   - Check that the link works
   - Ensure the markdown code is correct
5. **Submit a Pull Request** with:
   - A clear description of the new button
   - The SVG file
   - Updated README.md section

### SVG Guidelines

- **Format**: Valid SVG
- **Size**: Consistent with existing buttons (approximately 200x50px)
- **Style**: Modern, clean, and professional
- **Colors**: Use platform brand colors when appropriate
- **Optimization**: Remove unnecessary elements, optimize paths

### Code Style

- Use consistent naming conventions
- Follow the existing README format
- Keep descriptions concise and clear

### Pull Request Process

1. Update the README.md with your new button
2. Ensure all files are properly formatted
3. Test that the button displays correctly
4. Update the button count badge if needed
5. Write a clear PR description

## Questions?

If you have any questions, feel free to open an issue for discussion.
